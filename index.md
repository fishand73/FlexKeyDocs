# 灵动键 FlexKey 文档

灵动键是一款面向 Android 的自动化与快捷触发工具。它把“条件”和“动作”组合成规则：当按键、悬浮球、手势、通知或无障碍事件满足条件时，自动执行一个动作。

这份文档按当前代码实现整理，覆盖 App 的核心概念、内置功能、自定义动作、自定义条件脚本和 JavaScript API。

## 核心概念

一条规则由三部分组成：

- 条件：什么时候触发，例如按键单击、悬浮球滑动、通知出现、无障碍事件匹配或 JS 脚本返回 `true`。
- 生效范围：全局生效，或只在指定应用前台时生效。
- 动作：触发后执行什么，例如返回、打开应用页面、模拟按键、点击通知按钮、执行 JS 脚本。

## 能力与权限

灵动键依赖 Android 系统服务提供自动化能力。不同功能会按需提示依赖状态：

| 能力 | 用途 |
| --- | --- |
| 无障碍服务 | 按键/组合按键条件、无障碍事件条件、全局返回/主页/最近任务、无障碍节点动作、部分触控模拟 |
| 通知服务 | 通知事件条件、通知类 JS 条件、通知控件动作 |
| 悬浮窗权限 | 悬浮球条件和悬浮球展示 |
| Shizuku 授权 | 按键/触控模拟增强、截图、强制停止应用等 |
| 修改系统设置 | 自动旋转、亮度、方向等系统设置动作 |

按键模拟与触控模拟满足“无障碍服务或 Shizuku 授权”其中之一即可使用。部分系统或厂商权限没有公开判断 API，设置页会提供跳转入口并提示手动确认。

## 典型用法

1. 在“设置 > 服务与权限”开启需要的服务。
2. 在“动作”页安装内置动作，或创建自定义动作/JS 动作。
3. 在“规则”页创建规则，只选择一种条件类型。
4. 设置生效范围，选择动作并保存。
5. 在规则列表中启用、停用、长按拖动排序，或通过菜单更改动作/删除规则。
